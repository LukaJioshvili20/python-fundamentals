# python-fundamentals
 Synopsis of Python 3.10 for university 

## Deployment

To deploy pygames and run

For python 2
```bash
  python -m pip install --user pygame
```

For python 3
```bash
  python3 -m pip install --user pygame
```

For macOS , remove "--user" flag
```bash
  python3 -m pip install pygame
```
